애드센스 마케팅 상품 접수 시스템 - 완벽 사용설명서

이 문서는 기술적 지식 없이도 누구나 시스템을 사용할 수 있도록 쉽게 설명합니다.


============================================================================
1. 시스템이 무엇인가?
============================================================================

애드센스 마케팅 상품 접수 시스템은 간단히 말해 온라인 주문 시스템입니다.

생각해보세요:
- 마켓에서 물건을 사고 포인트를 사용하듯이, 여기서는 마케팅 상품을 신청하고 포인트를 차감합니다
- 관리자는 마치 마켓 운영자처럼 상품 가격을 정하고 고객(거래처) 계정을 관리합니다
- 거래처는 마치 마켓 고객처럼 로그인해서 상품을 신청하고 자신의 주문 내역을 봅니다

4가지 상품이 있습니다:
1. 플레이스 유입 (네이버 플레이스에서 더 많은 사람이 방문하게 하는 서비스)
2. 영수증 리뷰 (고객들이 영수증 기반으로 리뷰를 달게 하는 서비스)
3. 카카오맵 리뷰 (카카오맵에 좋은 평가를 남기게 하는 서비스)
4. 블로그 배포 (회사 블로그에 홍보 글을 올리는 서비스)


============================================================================
2. 첫 화면 - 로그인
============================================================================

컴퓨터 주소창에 시스템 주소를 입력하면 처음 보이는 화면입니다.

화면 설명:
- 중앙에 큰 "AdSense" 글자와 "마케팅 상품 접수 시스템" 설명이 있습니다
- 아래에 두 개의 탭이 있습니다: "거래처" 탭과 "관리자" 탭

거래처로 로그인하기:
1. "거래처" 탭을 클릭합니다 (이미 선택되어 있으면 그냥 진행)
2. 첫 번째 상자에 관리자로부터 받은 아이디를 입력합니다
3. 두 번째 상자에 비밀번호를 입력합니다
4. 파란색 "로그인" 버튼을 클릭합니다
5. 시스템이 인증하는 동안 "로그인 중..." 메시지가 보입니다
6. 성공하면 거래처 메인 화면으로 이동합니다

관리자로 로그인하기:
1. "관리자" 탭을 클릭합니다
2. 첫 번째 상자에 관리자 아이디를 입력합니다 (처음 설정 시 기본값: admin)
3. 두 번째 상자에 비밀번호를 입력합니다 (처음 설정 시 기본값: admin123)
4. 파란색 "로그인" 버튼을 클릭합니다
5. 관리자 메인 화면으로 이동합니다

로그인 실패 시:
- 빨간색으로 "아이디 또는 비밀번호가 틀렸습니다" 같은 오류 메시지가 나타납니다
- 아이디와 비밀번호를 다시 확인해서 입력하세요
- 계속 실패하면 관리자에게 아이디와 비밀번호를 다시 확인해달라고 요청하세요


============================================================================
3. 관리자가 할 수 있는 모든 것
============================================================================

관리자 로그인 후 왼쪽 메뉴에 여러 메뉴가 있습니다.
각 메뉴를 클릭하면 그 기능을 사용할 수 있습니다.


--- 3-1. 대시보드 (관리자 메인 화면) ---

왼쪽 메뉴에서 "대시보드" 또는 상단의 로고를 클릭하면 메인 화면이 나타납니다.

화면에 표시되는 정보:
- 총 거래처: 현재 몇 개의 거래처 계정이 있는지
- 대기 중인 접수: 아직 처리하지 않은 상품 신청이 몇 개인지
- 총 포인트: 모든 거래처가 가진 포인트의 합계
- 상품 카테고리: 현재 판매 중인 상품이 몇 가지인지

아래쪽에는 "최근 접수 내역"이라는 표가 있습니다.
이것은 가장 최근에 거래처들이 신청한 상품들의 목록입니다.


--- 3-2. 거래처 관리 ---

왼쪽 메뉴에서 "거래처 관리"를 클릭합니다.

화면에는 현재 등록된 거래처 목록이 표로 나타납니다.
표에는 다음 정보가 표시됩니다:
- 아이디: 거래처의 로그인 아이디
- 회사명: 거래처의 회사 이름
- 포인트: 해당 거래처가 현재 가진 포인트
- 생성 날짜: 계정을 만든 날짜
- 상태: 활성(사용 가능) 또는 비활성(사용 불가)

각 거래처 항목의 오른쪽에는 몇 개의 작은 버튼이 있습니다:
- 가격 설정: 이 거래처가 볼 수 있는 상품 가격을 정합니다
- 포인트 관리: 이 거래처의 포인트를 충전하거나 차감합니다
- 비밀번호 초기화: 거래처가 비밀번호를 잊었을 때 새 비밀번호를 설정합니다
- 수정: 거래처의 회사명이나 아이디 정보를 바꿉니다
- 삭제: 거래처 계정을 완전히 삭제합니다

거래처 추가하기:
1. 오른쪽 위에 파란색 "거래처 추가" 버튼을 클릭합니다
2. 팝업 창이 뜹니다
3. 다음을 입력합니다:
   - 아이디: 거래처가 로그인할 때 사용할 아이디 (예: naver_store_001)
   - 비밀번호: 거래처가 로그인할 때 사용할 비밀번호 (예: password123)
   - 회사명: 거래처의 회사 이름 (예: 서울 카페)
   - 초기 포인트: 이 거래처가 처음 가질 포인트 (예: 10000)
4. 팝업 하단의 "거래처 추가" 버튼을 클릭합니다
5. 성공하면 팝업이 닫히고 목록에 새 거래처가 추가됩니다

거래처 정보 수정하기:
1. 수정할 거래처의 "수정" 버튼을 클릭합니다
2. 팝업이 뜨고 현재 정보가 표시됩니다
3. 바꾸고 싶은 항목을 수정합니다
4. 팝업 하단의 "저장" 버튼을 클릭합니다
5. 변경사항이 저장됩니다

거래처 비밀번호 초기화하기:
1. 비밀번호를 초기화할 거래처의 "비밀번호 초기화" 버튼을 클릭합니다
2. 팝업이 뜹니다
3. 새로운 비밀번호를 입력합니다
4. 팝업의 "초기화" 버튼을 클릭합니다
5. 완료되면 그 거래처의 비밀번호가 변경됩니다

거래처 삭제하기:
1. 삭제할 거래처의 "삭제" 버튼을 클릭합니다
2. "정말 삭제하시겠습니까?" 확인 팝업이 뜹니다
3. "삭제" 버튼을 클릭해서 확인합니다
4. 해당 거래처가 완전히 삭제됩니다 (되돌릴 수 없음)


--- 3-3. 거래처별 상품 가격 설정 ---

거래처 관리 화면에서 각 거래처의 "가격 설정" 버튼을 클릭합니다.

나타나는 화면 설명:
- 해당 거래처가 이용 가능한 상품들이 나열됩니다
- 각 상품 옆에 가격을 입력하는 상자가 있습니다

상품 가격 설정하기:
1. 수정하고 싶은 상품의 가격 상자를 클릭합니다
2. 현재 가격을 지우고 새 가격을 입력합니다 (예: 5000)
3. 그 상품의 "표시" 체크박스가 있으면:
   - 체크되어 있으면: 이 거래처가 그 상품을 볼 수 있음
   - 체크 해제되어 있으면: 이 거래처가 그 상품을 볼 수 없음
4. 화면 하단의 "저장" 버튼을 클릭합니다
5. 모든 설정이 저장됩니다


--- 3-4. 상품 관리 ---

왼쪽 메뉴에서 "상품 관리"를 클릭합니다.

현재 시스템에는 4가지 기본 상품이 있고, 이들은 변경할 수 없습니다:
1. 플레이스 유입 접수
2. 영수증 리뷰 접수
3. 카카오맵 리뷰 접수
4. 블로그 배포 접수


--- 3-5. 포인트 관리 ---

왼쪽 메뉴에서 "포인트 관리"를 클릭합니다.

화면에는 모든 거래처의 포인트 현황이 표로 나타납니다.

표 설명:
- 거래처명: 회사 이름
- 보유 포인트: 현재 가진 포인트
- 소비 포인트: 상품을 신청할 때 사용한 포인트의 총합
- 행동: 포인트를 조정하는 버튼

포인트 충전/차감하기:
1. 수정할 거래처의 "관리" 버튼을 클릭합니다
2. 팝업이 뜹니다
3. 다음을 입력합니다:
   - 변경 유형: "충전" (추가) 또는 "차감" (빼기) 또는 "환불" (돌려주기)을 선택
   - 포인트: 변경할 포인트 금액 (예: 5000)
   - 사유: 왜 변경하는지 설명 (예: "초기 충전" 또는 "계약 종료로 환불")
4. 팝업 하단의 "저장" 버튼을 클릭합니다
5. 포인트가 변경되고 거래처의 기록에 남겨집니다


--- 3-6. 전체 접수 내역 조회 ---

왼쪽 메뉴에서 "접수 내역"을 클릭합니다.

화면에는 모든 거래처가 신청한 상품 목록이 표로 나타납니다.

표의 각 열:
- 날짜: 상품을 신청한 날짜와 시간
- 거래처: 신청한 거래처의 회사명
- 상품: 신청한 상품의 종류 (플레이스/영수증/카카오맵/블로그)
- 상품 정보: 신청한 상품의 세부 내용
- 포인트: 이 신청에 사용된 포인트
- 상태: 처리 대기 중, 완료 등


--- 3-7. 리포트 관리 ---

왼쪽 메뉴에서 "리포트"를 클릭합니다.

여기서는 생성된 리포트 파일 목록을 볼 수 있습니다.


--- 3-8. AS 요청 처리 ---

왼쪽 메뉴에서 "AS 신청"을 클릭합니다.

거래처에서 문제가 있다고 신청한 항목들의 목록이 나타납니다.


============================================================================
4. 거래처(일반 사용자)가 할 수 있는 모든 것
============================================================================

거래처로 로그인한 후 화면 설명입니다.


--- 4-1. 대시보드 (거래처 메인 화면) ---

거래처 로그인 후 처음 보이는 화면입니다.

화면 상단:
- 회사명이 표시됩니다
- 현재 가진 포인트가 큰 글씨로 보입니다 (예: "10,000 P")

중앙에는 4개의 상품 카드가 나타납니다.
각 카드에는:
- 상품 아이콘
- 상품 이름
- 간단한 설명
- "접수하기" 버튼
- 단가 (이 거래처가 그 상품을 신청할 때 사용되는 포인트)

아래쪽에는 "최근 접수 내역"이 나타납니다.


--- 4-2. 상품 신청하기 ---

메인 화면의 상품 카드 중 원하는 상품의 "접수하기" 버튼을 클릭합니다.

플레이스 유입 신청하기:
1. "플레이스 유입 접수" 카드의 "접수하기" 버튼을 클릭합니다
2. 상세 신청 화면으로 이동합니다
3. 다음을 입력합니다:
   - 신청 수량: 몇 개를 신청할 것인가 (예: 10)
   - 가게명: 신청하는 가게의 이름
   - 가게 주소: 가게의 주소
   - 추가 요청사항: 특별히 요청할 사항이 있으면 입력 (선택)
4. 화면 하단의 파란색 "접수하기" 버튼을 클릭합니다
5. 확인 팝업이 뜹니다: "정말 신청하시겠습니까? 포인트가 차감됩니다"
6. "확인" 버튼을 클릭합니다
7. 신청 포인트가 자동으로 차감되고 신청이 접수됩니다

영수증 리뷰/카카오맵 리뷰/블로그 배포도 비슷하게 진행됩니다.


--- 4-3. 접수 내역 조회 ---

왼쪽 메뉴에서 "접수 내역"을 클릭합니다.

화면에 이 거래처가 지금까지 신청한 모든 상품의 목록이 표로 나타납니다.


--- 4-4. 포인트 현황 ---

왼쪽 메뉴에서 "포인트"를 클릭합니다.

화면에 다음 정보가 표시됩니다:
- 현재 보유 포인트: 지금 가진 포인트
- 총 충전 포인트: 관리자로부터 받은 포인트의 총합
- 총 사용 포인트: 상품 신청에 사용한 포인트의 총합


--- 4-5. AS 신청 ---

왼쪽 메뉴에서 "AS"를 클릭합니다.

화면에 AS(문제 신청) 폼이 나타납니다.

AS 신청하기:
1. "신청 제목" 상자에 문제의 간단한 제목을 입력합니다
2. "상세 내용" 상자에 문제를 상세히 설명합니다
3. "신청 유형"에서 문제의 종류를 선택합니다
4. 화면 하단의 파란색 "AS 신청" 버튼을 클릭합니다
5. "신청이 완료되었습니다"라는 메시지가 나타나면 관리자가 검토해줄 것입니다


============================================================================
5. 화면 네비게이션
============================================================================

모든 페이지의 좌측 상단을 보면 메뉴가 있습니다.

관리자 메뉴:
- 대시보드
- 거래처 관리
- 상품 관리
- 접수 내역
- 포인트 관리
- 리포트
- AS 신청
- 분석

거래처 메뉴:
- 대시보드
- 상품 접수
- 접수 내역
- 포인트
- AS

로그아웃:
화면 오른쪽 위에 사용자 이름이 표시됩니다.
그 옆의 화살표를 클릭하면 "로그아웃" 옵션이 나타납니다.


============================================================================
6. 실제 사용 시나리오
============================================================================

새로운 카페가 서비스를 신청하는 과정:

1단계 - 관리자가 거래처 추가
- 거래처 관리 -> 거래처 추가 클릭
- 아이디, 비밀번호, 회사명, 초기 포인트 입력
- 추가 완료

2단계 - 관리자가 상품 가격 설정
- 거래처 관리에서 해당 거래처의 "가격 설정" 클릭
- 각 상품의 가격 입력
- 표시 여부 설정
- 저장 클릭

3단계 - 거래처가 로그인
- "거래처" 탭에서 아이디와 비밀번호 입력
- 로그인 클릭
- 메인 화면 확인

4단계 - 거래처가 상품 신청
- 원하는 상품의 "접수하기" 클릭
- 필요한 정보 입력
- 포인트 차감 확인 후 신청
- 신청 완료

비밀번호 관리:

자주 하는 실수:
1. 거래처 삭제를 실수로 하기 - 신중하게 사용하세요

3. 상품 가격을 너무 높게 설정 - 시장에 맞게 설정하세요
4. 거래처가 신청한 후 포인트를 또 차감하기 - 자동으로 차감됩니다
