거래처를 위한 빠른 시작 가이드

이 문서는 거래처 담당자가 시스템을 처음 사용할 때 필요한 기본 정보만 정리했습니다.


1. 로그인 방법
==================================================

1. 브라우저 주소창에 시스템 주소를 입력합니다
2. 로그인 화면이 나타나면 "거래처" 탭을 선택합니다
3. 관리자로부터 받은 아이디를 입력합니다
4. 비밀번호를 입력합니다
5. "로그인" 버튼을 클릭합니다
6. 메인 화면이 나타나면 로그인 성공입니다

* 아이디나 비밀번호가 틀리면 빨간 오류 메시지가 나타납니다
* 계속 실패하면 관리자에게 연락하세요


2. 메인 화면 보기
==================================================

로그인 후 나타나는 화면:

- 화면 상단: 당신의 회사명과 현재 보유 포인트가 표시됩니다
- 화면 중앙: 4개의 상품 카드가 있습니다 (플레이스, 영수증, 카카오맵, 블로그)
- 화면 하단: 최근에 신청한 상품들 목록

각 상품 카드에는:
- 상품 이름
- 상품 설명
- 단가 (예: 5000 P)
- "접수하기" 버튼


3. 상품 신청하기 (4가지 방법)
==================================================

플레이스 유입 신청:
1. "플레이스 유입 접수" 카드의 "접수하기" 버튼 클릭
2. 신청 수량 입력 (예: 10)
3. 가게명 입력 (예: 강남 커피숍)
4. 가게 주소 입력
5. 필요하면 추가 요청사항 입력 (선택)
6. "접수하기" 버튼 클릭
7. 확인 팝업에서 "확인" 클릭
8. 완료! 포인트가 자동으로 차감됩니다

영수증 리뷰 신청:
1. "영수증 리뷰 접수" 카드의 "접수하기" 버튼 클릭
2. 신청 수량 입력
3. 가게명 입력
4. 필요하면 추가 요청사항 입력
5. "접수하기" 버튼 클릭
6. 완료!

카카오맵 리뷰 신청:
1. "카카오맵 리뷰 접수" 카드의 "접수하기" 버튼 클릭
2. 신청 수량 입력
3. 가게명 입력
4. 필요하면 추가 요청사항 입력
5. "접수하기" 버튼 클릭
6. 완료!

블로그 배포 신청:
1. "블로그 배포 접수" 카드의 "접수하기" 버튼 클릭
2. 신청 수량 입력
3. 블로그 글 제목 입력
4. 블로그 글 내용 입력
5. 필요하면 추가 요청사항 입력
6. "접수하기" 버튼 클릭
7. 완료!

중요: 신청이 완료되면 포인트가 즉시 차감됩니다. 신청 전에 반드시 확인하세요!


4. 메뉴 사용 방법
==================================================

화면 왼쪽에 메뉴가 있습니다:

대시보드: 메인 화면으로 돌아가기

상품 접수: 상품을 신청하는 페이지로 이동
- 4가지 상품의 카드가 표시됩니다
- "접수하기" 버튼을 클릭하면 신청 폼이 나타납니다

접수 내역: 지금까지 신청한 모든 상품의 기록을 봅니다
- 날짜, 상품, 수량, 포인트, 내용이 표로 표시됩니다

포인트: 포인트 현황을 봅니다
- 현재 보유 포인트
- 총 받은 포인트
- 총 사용 포인트
- 포인트 거래 기록

AS: 문제가 있을 때 신청합니다
- 문제 제목 입력
- 상세 설명 입력
- 문제 유형 선택
- "AS 신청" 버튼 클릭
- 관리자가 검토하고 답장해줄 것입니다


5. 포인트 관리하기
==================================================

포인트란?
- 상품을 신청할 때 사용하는 "돈" 개념입니다
- 상품마다 정해진 포인트를 소비합니다
- 포인트가 부족하면 상품을 신청할 수 없습니다

포인트 확인:
- 메인 화면 상단에 큰 글씨로 표시됩니다
- 포인트 메뉴에서 자세한 기록을 볼 수 있습니다

포인트 부족 시:
- 관리자에게 포인트 충전을 요청하세요
- 관리자가 포인트를 추가해줄 것입니다

중요: 신청하면 취소할 수 없으므로 신청 전에 꼭 확인하세요!


6. 접수 내역 확인하기
==================================================

"접수 내역" 메뉴 클릭:

표에 다음 정보가 표시됩니다:
- 신청 날짜와 시간
- 신청한 상품 종류
- 신청한 개수
- 사용한 포인트
- 가게명, 주소 등 상세 정보

표 필터링:
- 상단의 필터에서 특정 상품만 볼 수 있습니다
- 날짜 범위를 선택할 수 있습니다

다운로드:
- "다운로드" 버튼을 클릭하면 엑셀 파일로 저장할 수 있습니다
- 이 파일을 다른 곳에 공유하거나 보관할 수 있습니다


7. 비밀번호 변경 (예정)
==================================================

현재: 비밀번호 변경 기능이 없습니다
- 비밀번호를 변경하려면 관리자에게 요청하세요
- 관리자가 새로운 비밀번호를 설정해줄 것입니다


8. 로그아웃하기
==================================================

화면 오른쪽 위에서:
1. 당신의 이름 옆 화살표를 클릭합니다
2. "로그아웃" 옵션을 클릭합니다
3. 로그인 화면으로 돌아갑니다

* 작업을 마친 후 반드시 로그아웃하세요 (보안을 위해)


9. 자주 하는 질문
==================================================

Q: 신청을 취소할 수 있나요?
A: 아니요. 신청하면 즉시 포인트가 차감되고 취소할 수 없습니다.
   신청 전에 꼭 확인하세요.

Q: 포인트가 부족합니다. 어떻게 하나요?
A: 관리자에게 포인트 충전을 요청하세요.
   관리자가 당신의 계정에 포인트를 추가해줄 것입니다.

Q: 로그인이 안 됩니다. 어떻게 하나요?
A: 다음을 확인하세요:
   1. 아이디 오른쪽에 공백이 없는가?
   2. 비밀번호가 정확한가?
   3. "거래처" 탭을 선택했는가?
   계속 실패하면 관리자에게 연락하세요.

Q: 신청한 내용을 봤는데 정보가 틀렸습니다. 수정할 수 있나요?
A: 신청 내용은 수정할 수 없습니다.
   관리자에게 AS 신청을 해서 문제를 보고하세요.

Q: 신청한 상품이 처리되었는지 어떻게 알 수 있나요?
A: "접수 내역"에서 신청한 상품의 상태를 확인할 수 있습니다.

Q: 포인트 기록을 인쇄하고 싶습니다.
A: "접수 내역"에서 "다운로드" 버튼을 클릭하면 엑셀 파일로 저장할 수 있습니다.
   엑셀에서 인쇄할 수 있습니다.


10. 중요한 주의사항
==================================================

반드시 기억하세요:

1. 신청은 취소할 수 없습니다
   - 신청 전에 3번 확인하세요
   - 실수로 신청해도 포인트는 돌아오지 않습니다

2. 포인트는 관리자만 충전할 수 있습니다
   - 포인트가 부족하면 신청할 수 없습니다
   - 미리미리 충전을 요청하세요

3. 비밀번호는 안전하게 보관하세요
   - 다른 사람에게 비밀번호를 알려주지 마세요
   - 계정을 함께 사용하지 마세요

4. 문제가 생기면 바로 관리자에게 연락하세요
   - "AS 신청" 메뉴를 사용하거나
   - 관리자에게 직접 전화하세요

5. 신청하기 전에 항상 확인하세요
   - 신청 수량이 맞는가?
   - 포인트가 충분한가?
   - 가게명, 주소가 정확한가?


11. 빠른 체크리스트
==================================================

시스템을 처음 사용할 때:

□ 아이디와 비밀번호 확인했습니다
□ 로그인했습니다
□ 보유 포인트를 확인했습니다
□ 상품 단가를 확인했습니다
□ 신청할 상품을 결정했습니다
□ 신청 정보를 입력했습니다
□ 사용할 포인트가 맞는지 확인했습니다
□ "접수하기" 버튼을 클릭했습니다
□ 확인 팝업에서 다시 한 번 확인했습니다
□ "확인"을 클릭했습니다
□ "신청이 완료되었습니다" 메시지를 확인했습니다
