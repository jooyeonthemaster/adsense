③ 각 섹터 별 상세 내용
* 각 카테고리 대시보드 내 최상단 부분에 공지사항, 가이드, 유의사항 등등 관리자가 자유 변경 할 수 있도록 깔끔하게 위치 희망

* 리워드/리뷰마케팅/블로그배포는 [AS 신청] 기능 고정 활성화, 모두 동일하게 작업 수량 대비 누락률 20%은 신청 버튼 비활성화

* 각 ID별 셋팅된 상품 별 단가로 신청하는 건에 맞춰 포인트 차감되는 형식으로 진행

* 리워드 제외하고 모든 상품들은 작업되는 중간중간 [리포트] 자유롭게 엑셀파일 형식으로 다운로드 될 수 있게 하는 기능 희망 (리포트는 관리자가 등록해두는 방향으로 하겠습니다)

- 접수현황 확인 시 [확인중] [접수완료] [구동중] 과 같이 현재 작업 상태에 대해 한 눈에 볼 수 있는 표시바 삽입 희망

- 표시바 우측에 [중단] 삽입하고, 클릭 시 '작업 중단은 가능하나, 환불은 어렵다' 와 같은 경고 팝업이 뜨고 체크박스로 '동의' 클릭 시 '중단 신청이 완료되었습니다' 로 마무리 → '동의하지 않습니다' 클릭 시 중단 요청 안되게끔 진행

✔ 리뷰 마케팅
'방문자 리뷰(아이콘)' 'K맵 리뷰(아이콘)' 위치 후 클릭형

⑴ '방문자 리뷰' 선택 → 업체명/플레이스 링크 삽입(삽입 시 링크 MID값 자동으로 불러와지게) → 일 발행수량 선택(최소 1건~최대 10건) → 총 작업일 기재(기재 후 총 작업수량 자동계산되어 우측에 표기) → 체크박스로 사진유무 / 지정원고 체크(중복체크 O) → 가이드 및 요청사항 기재란(장문텍스트) → 첨부파일 삽입 기능(사업자등록증 or 샘플 영수증을 받을 예정이라 JPG or PNG or PDF 파일 허용) → 접수 신청

- 접수현황 확인 시 [확인중] [접수완료] [구동중] 과 같이 현재 작업 상태에 대해 한 눈에 볼 수 있는 표시바 삽입 희망

- 표시바 우측에 [중단] 삽입하고, 클릭 시 팝업 안내멘트(ex. 이미 예약 구동된 수량 제외 남은 건에 대해 환불이 진행됩니다) 팝업이 뜨고 체크박스로 '동의' 클릭 시 '중단 신청이 완료되었습니다' 로 마무리 → '동의하지 않습니다' 클릭 시 중단 요청 안되게끔 진행

⑵ 'K맵 리뷰' 선택 → 업체명/카카오맵 링크 삽입 → 일 발행수량 입력(최소 1건~최대 MAX없음) → 총 작업일 기재(기재 후 총 작업수량 자동 계산되어 우측에 표기) → 체크박스로 사진유무 / 지정원고 or AI제작원고 체크(or 빼고는 중복체크 O) → 사진 비율 선택(10%~100%) → 별점 선택(체크박스로 4~5점대 섞어서 or 5점대만 or 4점대만 옵션형) 가이드 및 요청사항 기재란(장문텍스트) → 접수 신청

- 접수 받은 후 작업 될 '이미지'와 '원고'를 관리자가 등록할 수 있는 기능 삽입

- 등록 된 작업 예정 리스트 유저 확인 및 수정요청 버튼 → '검수완료 or 수정요청' 두개의 버튼 노출 → '검수완료' 시 '구동이 시작됩니다' 와 같은 팝업 노출 → '수정요청' 시 장문 텍스트로 코멘트 남길 수 있도록 하고 [확인] 버튼으로 마무리 (ex. 원고에 이모티콘 빼주세요) 

- '수정요청' 은 메일의 전달 기능처럼 왕복으로 주고받는 메시지 히스토리 노출 희망

⑴ '블로그(아이콘)' 선택 → 업체명/플레이스 링크 삽입(배송형의 경우 상품링크 삽입) → 제공내역 기재(단문텍스트) → 희망 팀수 입력 → 방문가능요일 체크 → 방문가능시간대 선택(ex. 오전 11시 ~ 오후 9시) → 작성 시 참고 가이드라인(장문텍스트) → 포스팅 내 희망 키워드(메인:지역키워드 / 서브:분류키워드 나눠서 작성되게 / 최대 5개 / ex. 메인:강남 / 서브:맛집 + 메인:강남역 / 서브:샤브샤브..)

- 신청 후 2주 이내 신청된 블로거들의 리스트를 관리자가 기재할 수 있도록 구현 (이름/블로그주소/블로그지수)

- 업데이트 된 리스트 내에서 유저가 체크하여 리스트를 다시 관리자 측으로 넘길 수 있게끔 구현

- 넘어온 후엔 기재 창 추가 (방문희망일/시간/방문인원) 하여 최종 체크 된 인원들을 유저에게 다시 전달

- 유저는 정보를 보고 [조율요청] 가능하나, 요청 시 1:1 고객센터(링크이동)로 문의하여 CS할 수 있도록 희망

- 각 인원별 발행 이후에 '키워드 노출 현황'에 관리자가 키워드+순위 기재할 수 있는 탭 추가 생성 (ex. 강남역 샤브샤브 / 13위)

- 선정된 모든 인원들의 '발행링크' 가 다 채워지면 캠페인 종료

✔ 카페침투 마케팅
'카페침투(아이콘)' 위치

⑴ '카페침투(아이콘)' 선택 → 업체명/플레이스 링크 삽입 → 종류 선택(후기성 or 정보성 체크박스) → 지역군 선택 시 지역별로 나눠놓은 지역별 리스트들 노출 희망 → 리스트 중 희망 카페 선택(중복 O) → 발행 건수 선택 → 가이드 및 요청사항(장문 텍스트) 기재 → 사진유무 체크박스 선택 → 접수신청

- 접수현황 확인 시 [확인중] [접수완료] [원고작성중] [원고작업완료] [구동중] 과 같이 현재 작업 상태에 대해 한 눈에 볼 수 있는 표시바 삽입 희망

- [원고작업완료] 뜰 경우 링크(구글스프레드시트)로 이동될 수 있게끔 하이퍼링크 삽입

- 표시바 우측에 [중단] 삽입하고, 클릭 시 팝업 안내멘트(ex. 이미 예약 구동된 수량 제외 남은 건에 대해 환불이 진행됩니다) 팝업이 뜨고 체크박스로 '동의' 클릭 시 '중단 신청이 완료되었습니다' 로 마무리 → '동의하지 않습니다' 클릭 시 중단 요청 안되게끔 진행
